List of 3DS homebrew

This is a full list containing all homebrew software released for the Nintendo 3DS.
I'll try to keep it up-to-date whenever possible, feel free to contribute if you want to.
For a list of all existing exploits, see here.
You can search homebrew by typing part of the name, description, author, release date, compatible entrypoints, or any other keyword.
For example: "wifi" will return a list of homebrew that uses wifi, "emulators" will return a list of emulators, etc.
You can also sort by name, description, author and release date.

Formats:
- 3DSX: used by the homebrew launcher, which requires ninjhax/menuhax/browserhax/ironhax.
- CIA: can be installed to and launched from the 3DS's home menu.
- 3DS: launched through the Gateway flashcard.
- DAT: launched through the internet browser, by accessing go.gateway-3ds.com, or through cakehax.
- BIN: These can be either ARM9 payloads, launched through BrahmaLoader or arm9loaderhax; or ARM11 payloads, launched by the internet browser via spiderhax.
